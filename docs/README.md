## üîë Key Information

### API Credentials
- **Environment:** Production/Sandbox
- **Default Env:** Sandbox (for testing)
- **Auth Method:** JWT Bearer Token + x-api-key header

### Database
- **Database:** tds_autofile
- **User:** tdsuser
- **Credentials Table:** api_credentials (firm_id, environment, api_key, api_secret, access_token)

### File Locations
- **API Integration:** `/tds/lib/SandboxTDSAPI.php`
- **Dashboard Pages:** `/tds/admin/*.php`
- **Configuration:** `/tds/config.php`
- **Database Setup:** `/tds/lib/db.php`
