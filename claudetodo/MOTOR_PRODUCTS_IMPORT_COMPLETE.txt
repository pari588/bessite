================================================================================
SPECIFICATIONS DATABASE STRUCTURE
================================================================================

Main Table: mx_motor
- motorID (primary key)
- motorTitle (full product name)
- motorSubTitle (short description)
- motorDesc (full description)
- motorImage (product image filename)
- categoryMID (category ID)
- seoUri (SEO-friendly URL)
- status (active/inactive flag)

Details Table: mx_motor_detail
- motorDID (primary key)
- motorID (foreign key to mx_motor)
- descriptionTitle (variant title: "Output | Voltage")
- descriptionOutput (kW/Hp rating)
- descriptionVoltage (voltage specification)
- descriptionFrameSize (frame size)
- descriptionStandard (certification/standard)
- status (active/inactive)

SUPPORT & MAINTENANCE
================================================================================

To add new motor products:
1. Log into admin panel (/xadmin/)
2. Navigate to Motors → Add New Motor
3. Fill in Title, Sub-Title, Description, Image
4. Select Category
5. Click "Add Motor Details" to add variants
6. Fill in Output, Voltage, Frame Size, Standard for each variant
7. Save and publish

To edit existing products:
1. Go to Motors → Motor List
2. Click Edit on the product
3. Modify any fields as needed
4. Update variants/specifications in the details section
5. Save changes

To manage categories:
1. Go to Motors → Motor Categories
2. View category hierarchy
3. Add new categories as needed
4. Assign products to correct categories
