## Selection Guide

### Choose Split Phase Motors if:
- Cost is primary concern
- Light-duty applications only
- Low starting torque acceptable
- Non-continuous operation
- Budget is critical

### Choose Capacitor Start Motors if:
- High starting torque needed
- Pump or compressor applications
- Equipment needs rapid acceleration
- Willing to pay for premium
- Agricultural or heavy-load use

### Choose Capacitor Run Motors if:
- Energy efficiency important
- Continuous operation planned
- Lower running cost desired
- Fan or blower applications
- Environmental compliance needed

### Choose PSC Motors if:
- Reliability is paramount
- Simplicity preferred
- Variable load operation expected
- Long service life needed
- Cost-effectiveness with reliability

## Database Implementation

### Table: mx_motor_detail
All specifications stored with status = 1 (active)

**Verification Queries**

```sql
-- Count specifications per motor
SELECT motorID, COUNT(*) as spec_count
FROM mx_motor_detail
WHERE motorID IN (48, 49, 50, 51)
GROUP BY motorID;

-- View all specs for Split Phase Motors
SELECT descriptionTitle, descriptionOutput, descriptionVoltage,
       descriptionFrameSize, descriptionStandard
FROM mx_motor_detail
WHERE motorID = 51
ORDER BY descriptionTitle;

-- Check total Single Phase specs
SELECT COUNT(*) as total_specs
FROM mx_motor_detail
WHERE motorID IN (48, 49, 50, 51);
```

## Wiring & Installation

### Typical Single Phase Motor Connections
- **Split Phase**: Start winding + Run winding + Centrifugal switch
- **Capacitor Start**: Capacitor + Start winding + Run winding + Switch
- **Capacitor Run**: Capacitor + Run winding (always connected)
- **PSC**: Capacitor + Single winding (permanent connection)

### Installation Considerations
- Proper grounding required
- Capacitor replacement intervals (Capacitor models)
- Ventilation for cooling
- Mounting on vibration-free base
- Proper electrical connection

## Troubleshooting

### Issue: Motor won't start
**Solutions**:
- Check power supply
- Verify capacitor (if equipped)
- Check centrifugal switch
- Test winding continuity

### Issue: Low torque
**Solutions**:
- For Split Phase: Not suitable for high-torque. Consider Cap Start
- Check voltage supply
- Verify capacitor value and condition
- Check bearing friction

### Issue: Motor overheating
**Solutions**:
- Check load (may be excessive)
- Verify ventilation
- Check for bearing issues
- Reduce duty cycle if light-duty model
